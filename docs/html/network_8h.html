Detailed Description

Common network connection utilities for NFS components.

This module provides common networking functionality used by both client and storage server components to connect to the naming server

◆ get_local_port()

int get_local_port ( int  sockfd)

Retrieves the local port number of a connected socket.

Parameters
sockfdThe socket file descriptor.
Returns
int The local port number on success, -1 on failure.

This function uses the getsockname system call to retrieve the local address and port number associated with a socket. It then extracts and returns the port number.

Note
The socket must be connected before calling this function.
